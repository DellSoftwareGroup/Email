We are excited to announce that Change Auditor 6.6 is now ready for download.

What's new in v6.6:

  • A new auditing module to audit cloud-based storage activity - IT departments today are at the whim of the latest cloud storage application or vendors. Users can easily take corporate data and store it on the cloud. To facilitate the management of this, the Dell Change Auditor for Cloud Storage auditing module automatically monitors the following cloud storage providers if installed: Dropbox™, Dropbox™ for Business, Box®, and OneDrive®.
  • Auto deployment Update - A streamlined deployment process that allows the ability to auto deploy workstation agents.
  • Supported upgrade path for Change Auditor 6.6 - Upgrading Change Auditor to version 6.6, requires the database version to be at least 6.0. To use an existing 5.x database, it must be first updated to the new schema by installing Change Auditor 6.0 or 6.5; alternatively, a new 6.6 database can be created and the required events can be moved from 5.x database.
